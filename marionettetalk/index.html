Testing Firefox OS Apps with Marionette

What is Marionette, Anyway??

  • based on the Webdriver JSON wire protocol
  • is an up and coming standard for browser testing
  • We support a Python based client and test runner

How does it work?

  • Based on a client/server model
  • Server listens on port for client commands
  • Server uses the MessageManager to talk to chrome/content
  • Designed with out-of-process communication in mind

How is it being used?

  • Test FirefoxOS Desktop and device builds
  • Test WebAPI's like SMS/ Telephony etc
  • Test FirefoxOS apps

How can I use it?

  • Marionette is enabled by default in:
    • http://ftp.mozilla.org/pub/firefox desktop debug build
    • http://ftp.mozilla.org/pub/b2g/ B2G desktop builds

  • By adding the following to user.js:
    user_pref("marionette.defaultPrefs.enabled", true);
    user_pref("marionette.force-local", true);
